# 三码合一工控端桌面程序

> An electron-vue project demo

#### 构建步骤

``` bash
# 安装依赖
npm install

# 在本地主机:8090上热重新加载服务
npm run dev

# 为生产构建electron应用程序
npm run build:dir

```
